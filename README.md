# QR-Code-Generator-Python-Project-
A simple Python project that generates QR Code images from a list of URLs using the qrcode library.

🚀 Project Description

This project automatically:

1- Takes a list of URLs (or emails/links)

2- Generates QR codes for each one

3- Saves them as image files (.png)

It is useful for:

1- Portfolio links

2- Social media sharing

3- Business cards

4- Marketing materials

🛠️ Tech Stack:

1- Python 🐍

2- qrcode library 📦

3- PIL (Pillow) for image generation

📦 Installation:

Install required library:
pip install qrcode[pil]

▶️ How It Works :
The project follows 5 main steps:

1- Import required library

2- Configure QR Code settings

3- Add data (URLs)

4- Build QR Code image

5- Save images as PNG files

🎯 Features:

1- 🔗 Generate QR codes from URLs

2- 🎨 Custom colors (red/white)

3- 📁 Save multiple QR images automatically

4- ⚡ Fast and lightweight
